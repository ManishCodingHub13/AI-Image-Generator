# 🎨 AI Image Generator

An AI-powered Image Generator that transforms text prompts into stunning and unique images. Simply describe what you want to see, and the AI generates an image based on your imagination.

This project demonstrates the integration of Artificial Intelligence with a user-friendly interface, allowing users to create artwork, illustrations, concept designs, and more from simple text descriptions.

## ✨ Features

- Generate images from text prompts
- Easy-to-use interface
- Fast image generation
- AI-powered creativity
- Download generated images
- Beginner-friendly project structure
- Responsive and modern design

## 🚀 How It Works

1. Enter a text prompt describing the image you want.
2. Click the **Generate** button.
3. The AI processes your prompt.
4. The generated image is displayed on the screen.
5. Save or download the image if desired.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Image-Generator.git
```

### 2. Navigate to the Project Directory

```bash
cd AI-Image-Generator
```

### 3. Install Dependencies

Frontend:

```bash
cd client
npm install
```

Backend:

```bash
cd ../server
npm install
```

### 4. Configure Environment Variables

Create a `.env` file in the server directory:

```env
API_KEY=your_api_key_here
```

### 5. Start the Backend Server

```bash
npm start
```

### 6. Start the Frontend

```bash
cd client
npm start
```

### 7. Open in Browser

```text
http://localhost:3000
```
